# Label

## Purpose
Label displays short, color-coded metadata or categorization information.

## When to use
Use to communicate status, category, or contextual metadata.

## When not to use
Do not use as an interactive control or navigation element.

## Design intent
Provide a compact, scannable visual indicator with semantic tone support.

## Accessibility
Labels should not be the only means of conveying critical information.
