# Focus Ring

## Purpose
Focus Ring provides a consistent, accessible keyboard focus indicator across interactive components.

## When to use
Use on all focusable interactive components.

## When not to use
Do not use as a standalone visual element or replace native focus without providing an equivalent indicator.

## Design intent
Centralize focus styling so all controls share the same appearance and behavior.

## Accessibility
Must appear only for keyboard-visible focus and meet contrast requirements.
