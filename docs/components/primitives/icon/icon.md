# Icon

## Purpose
Icon communicates meaning through a compact visual symbol and supports other components.

## When to use
Use alongside labels or where a universally recognized glyph improves comprehension.

## When not to use
Do not rely on icons alone to convey critical information unless an accessible name is provided.

## Design intent
Provide a consistent icon system with standardized sizing, color inheritance, and semantic usage.

## Accessibility
Decorative icons should be hidden from assistive technology. Informative icons require an accessible label or accompanying text.
